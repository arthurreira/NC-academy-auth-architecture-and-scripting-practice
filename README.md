# NordCloud Academy | Auth Architecture and Scripting Practice

> Project Type: Learning project (docs plus scripting exercises)

## Overview
This folder contains practice tasks and small prototypes related to frontend and backend development, plus basic file processing scripts.

Main items in this folder:
- Backend and frontend specification documents for an authentication and account management app.
- JavaScript exercises for array/object logic and file operations.
- Python and Node examples for reading and printing file contents.

## What We Did
- Wrote backend specification draft in [BACKEND.MD](BACKEND.MD) covering auth routes, account operations, and security/rate limiting expectations.
- Wrote frontend specification draft in [FRONTEND.MD](FRONTEND.MD) for login, signup, and dashboard flow with validation and error handling.
- Implemented JavaScript utility exercises in [python-test.py](python-test.py) and [test.js](test.js) to practice algorithmic thinking and collection handling.
- Practiced Node.js file I/O with [quality.js](quality.js) and XML reading with [test.js](test.js).

## What I Learned
- How to translate product requirements into API and UI specifications.
- How to structure authentication flows from signup/login to protected dashboard access.
- How to validate input and think about error states and user feedback.
- How to use file read/write APIs in both Node.js and Python.

## Tech Stack
- JavaScript (Node.js)
- Python
- Markdown documentation
- XML and plain text file handling
